# CS 4348 Project 3

GitHub Repository: https://github.com/skisurfer13/CS-4348-Project-3

## How to Run

This project is written in Python and uses only the Python standard library.

Run commands using:

```bash
python3 project3.py <command> <arguments>
```

## Test Commands

### 1. Create an index file

```bash
python3 project3.py create test.idx
```

### 2. Insert a key/value pair

```bash
python3 project3.py insert test.idx 15 100
```

### 3. Search for a key

```bash
python3 project3.py search test.idx 15
```

### 4. Load key/value pairs from a CSV file

```bash
python3 project3.py load test.idx input.csv
```

### 5. Print all key/value pairs

```bash
python3 project3.py print test.idx
```

### 6. Extract all key/value pairs to a CSV file

```bash
python3 project3.py extract test.idx output.csv
```
